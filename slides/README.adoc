== Generate slides:

```
SLIDES_FILE=
asciidoctor -T ./asciidoctor-reveal.js/templates/slim ${SLIDES_FILE}.adoc
asciidoctor -T ./asciidoctor-reveal.js/templates/slim ${SLIDES_FILE}.adoc && firefox ${SLIDES_FILE}.html &
```
